# Search and Filter User Guide

## How to Use the Search and Filter System

### 1. Searching from Home Page

**Step 1**: On the home page, find the search bar in the hero section
**Step 2**: Type your search query (e.g., "hostel near college", "PG in Sikar")
**Step 3**: Click the search button or press Enter
**Result**: You'll be taken to the Browse Properties page with your search applied

### 2. Exploring by City

**Step 1**: Scroll down to the "Cities We Serve" section on the home page
**Step 2**: Click "Explore Sikar" (or Jaipur/Kota) button on any city card
**Result**: Browse page opens with properties filtered by that city

- Each badge shows one active filter

**Remove Individual Filter**:
- Click the "√ó" button on any filter badge

### 6. Sorting Results

Choose from 5 sort options:
- **Newest**: Recently added properties first
- **Price: Low to High**: Budget-friendly options first
- **Price: High to Low**: Premium options first
- **Name: A-Z**: Alphabetical order
- **Name: Z-A**: Reverse alphabetical

### 7. View Modes

**Grid View** (Default):

**List View**:
- Full-width property cards

### 8. Mobile Experience

- Tap "Filters" button (shows active count)

**Applying Filters**:
- Make your selections

### 11. Common Search Scenarios

**Scenario 1: Budget Student Accommodation**
1. Click "Student" quick filter
2. Adjust price range if needed
3. Select your city
4. Done!

**Scenario 2: Premium Flat in Specific Area**
1. Select city (e.g., Jaipur)
2. Choose locality (e.g., Malviya Nagar)
3. Select "Flat" as property type
4. Click "Premium" quick filter
5. Browse results

**Scenario 3: PG with Specific Amenities**
1. Select "PG" as property type
2. Choose your city
3. Select required amenities (WiFi, AC, etc.)
4. Toggle "Food Included" if needed
5. Apply filters

**Scenario 4: Verified Properties Only**
1. Toggle "RoomSaathi Verified Only"
2. Add other filters as needed
3. All results will be verified

### 12. Understanding Filter Behavior

**Cumulative Filtering**:
- All active filters work together
- Results must match ALL selected criteria
- More filters = fewer but more relevant results

**Dynamic Updates**:
- Results update as you change filters
- No need to click "Apply" or "Search"
- Instant feedback

**Filter Persistence**:
- Filters stay active when you navigate
- Return to browse page with same filters
- Refresh page keeps filters (from URL)

### 13. Troubleshooting

**No Results Found?**
- Try removing some filters
- Broaden your price range
- Select different locality
- Use fewer amenity requirements

**Too Many Results?**
- Add more specific filters
- Narrow price range
- Select specific locality
- Add amenity requirements

**Filters Not Working?**
- Check if multiple conflicting filters are active
- Try clearing all and starting fresh
- Ensure city is selected before locality
